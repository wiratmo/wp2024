# PKL Monitoring
Intergarasi sistem pengajuan, verifikasi, monitoring, dan evaluasi PKL. 

### Installation

```bash
    composer install
    composer update
    php artisan key:generate
    php artisan cache:clear
    php artisan migrate
    php artisan db:seed

```
